# Chadwick Q2C Web Application

## Description
The Chadwick Q2C Web App is a modern system built to make quoting, costing, and tender preparation fast and accurate for electrical switchboards.  
It replaces slow spreadsheets with a simple, clean interface where users can build quotes, add boards, select items, and export a professional tender document.

## Features

### Quote Management
- Create, edit, duplicate, and delete quotes
- See all quotes in one dashboard with search and sorting
- Quick summary showing number of boards and total quote value
- Tag quotes with statuses like “Urgent”, “Amended”, or “Approved”

### Switchboard Builder
- Add multiple boards inside a quote
- Choose board type and enter configuration (IP rating, Form, Fault level, enclosure, etc.)
- Automatically load required basic items and busbars
- Add items from multiple catalogs (switchboards, basics, busbars, vendors)
- Real-time cost updates

### Pre-Selection Wizard
- Helps set key board specifications
- Controls bullet points and descriptions in the tender export
- Supports drawing references and board-specific notes

### Tender Export (Word Document)
- Generates a clean, formatted tender document
- One row per board with detailed bullet-point descriptions
- Automatically includes:
  - Switchboard specifications
  - Breakers, meters, CT metering, surge diverters, ATS/MTS
  - Notes and drawing references
- Uses a fixed Word template for consistent professional output

### Dashboard Enhancements
- Sorting by:
  - Recently updated
  - Quote number
  - Project name
  - Client name
  - Total value
- Hover actions for quick access:
  - Duplicate quote
  - Export tender
  - Delete quote

## Intended Users
- Estimators
- Project managers
- Sales teams
- Anyone preparing switchboard pricing and tender documentation

## Purpose
The goal of the system is to:
- Improve speed and accuracy
- Prevent manual errors
- Keep all quotes consistent
- Save time for estimators
- Produce professional tender documents automatically

## Notes
This system is built specifically for Chadwick Switchboards and is not for public use or distribution.
